# **App Name**: Curriculum Vitae Canvas

## Core Features:

- Real-Time CV Editing: Provide a split-screen interface with a live preview that updates as the user types.
- PDF Export: Implement a 'Download PDF' button that triggers the browser's print function to export the resume in A4 format.
- Data Input Forms: Enable scrollable form for data entry covering personal information, profile summary, education details, skills, and academic projects.
- CV Formatting Tool: An AI powered tool that can suggest relevant points the applicant should incorporate into their CV depending on their chosen career path.
- Data Persistence: Incorporate local storage to preserve the entered data.

## Style Guidelines:

- Primary color: Bright Purple (#6C63FF) to establish a playful and modern aesthetic.
- Accent color: Sunny Yellow (#FFD93D) for highlights and interactive elements to guide the user's focus.
- Background color: Soft Lavender (#F3F0FF) to provide a gentle, non-distracting backdrop.
- Headings font: 'Patrick Hand' (handwritten style, sans-serif) for a personal, approachable feel.
- Body text font: 'Poppins' (clean, rounded sans-serif) to ensure readability and modernity. Note: currently only Google Fonts are supported.
- Split-screen layout: Editor (40%) on the left, Live Preview (60%) on the right, to facilitate simultaneous editing and visualization.
- Deeply rounded corners (20px radius) for input boxes and buttons to soften the interface.
- Subtle organic blobs and soft gradients in the background to enrich the visual experience without being distracting.
